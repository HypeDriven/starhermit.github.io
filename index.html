SYS.02 // FOR CREATORS

Publish in minutes.
Own it forever.

No SDK to integrate before you ship. No platform approval queue. No build-upload ritual. If your game lives on GitHub, it's already one step away from being playable on StarHermit.

01

Paste a Repo

Add any GitHub repository URL. StarHermit ingests the repo, builds the game, and generates a public platform page — automatically. Anyone can do it: fans can surface abandoned open-source gems, archivists can preserve forgotten projects. Every page shows its source repository, so provenance is always transparent.

02

Claim Your Game

Link your GitHub account and StarHermit verifies you control the source repo. Then take the reins: customize artwork, trailers, screenshots, and tags; control builds, branches, and release channels (stable / beta / experimental); set pricing and permissions for your team.

03

Get Paid

Set a price, run sales, offer pay-what-you-want, or stay free with supporter tiers. Revenue flows directly to you — whether players pay in dollars, euros, or digital assets, it lands in one unified revenue stream.
